# Illinois CS Wiki

A student-maintained wiki for CS courses, guides, and resources at the University of Illinois Urbana-Champaign. Built and maintained by [ACM@UIUC's Academic Committee](https://academic.acm.illinois.edu).

## About

This wiki provides:
- **Course articles** - Student perspectives on CS, Math, Stats, ECE, and Physics courses
- **Guides** - Helpful tutorials and tips for navigating UIUC

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [Yarn](https://yarnpkg.com/)

### Installation

```bash
# Clone the repository
git clone https://github.com/acm-uiuc/wiki.git
cd wiki

# Install dependencies
yarn install

# Start the development server
yarn dev
```
